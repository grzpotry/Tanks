TANKS INSTRUCTIONS

Rules: Protect the flag and survive for remaining time in each stage
Win: Complete all stages
Lose: Lose all health or any flag is destroyed
Controls: WSAD or ARROWS + SPACE
Adding & configuring levels: To add new level to game just add new file in Resources\Scenes and name it accordingly as Level[X]

Planned features:
- game menu
- local coop
- random rewards (spawned on tank/wall destroyed)
- calculate score (for destroying tanks/walls)
- vfx & sfx support
- leaderboards
- more enemy types
- more game modes (now just protect the flag)
